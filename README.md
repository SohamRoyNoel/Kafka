# Starting Kafka
  - Start Zookeeper
  ```
  PS D:\Kafka\kafka_2.12-2.5.0> zookeeper-server-start.bat config\zookeeper.properties
  ```
  - Start Kafka
  ```
  PS D:\Kafka\kafka_2.12-2.5.0> kafka-server-start.bat config\server.properties
  ```
